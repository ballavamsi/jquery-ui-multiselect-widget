Basic Demos

Basic Without Option Group Checkbox

$(function(){
   $("#select1").multiselect().multiselectfilter(); 
});

Basic With Option Group Checkbox

Both multiselects are created with the following one-liner. Optgroup support is built in out of the box:

$(function(){
   $("#select2").multiselect({optGroupCheckbox : true}).multiselectfilter(); 
});

Basic With Option Group Checkbox and Treeview

Both multiselects are created with the following one-liner. Optgroup support is built in out of the box:

$(function(){
$("#select3").multiselect({optGroupCheckbox : true,treeView:true}).multiselectfilter(); 
});

Basic with only treeview

Both multiselects are created with the following one-liner. Optgroup support is built in out of the box:

$(function(){
$("#select4").multiselect({treeView:true}).multiselectfilter(); 
});

Collapse Treeview

$(function(){
$("#select10").multiselect({treeView:true,treeViewCollapseOnLoad:true}).multiselectfilter(); 
});


Search filter only Parent

Both multiselects are created with the following one-liner. Optgroup support is built in out of the box:

$(function(){
$("#select5").multiselect({optGroupCheckbox:true,treeView:true}).multiselectfilter({onlyOptGroupSearchable:true}); 
});

Include Parent and Child search

Both multiselects are created with the following one-liner. Optgroup support is built in out of the box:

$(function(){
$("#select6").multiselect({optGroupCheckbox:true,treeView:true}).multiselectfilter({optGroupSearchable:true}); 
});
